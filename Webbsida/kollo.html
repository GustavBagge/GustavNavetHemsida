Navet codar html

Kursen är uppdelad i tre delar precis som HTML-kodning. HTML som behandlar innehållet. CSS som låter oss bestämma hur webbsidan ska se ut och JS och bestämmer funktioner på sidan. Här hittar du digitala uppdragskort för alla delar.

Dag 1: HTML

Hyper Text Markup Language

HTML är inte ett programmeringsspråk utan snarare ett sätt att märka upp text och objekt. Genom att sätta ut olika markeringar kring ord, meningar och stycken så berättar vi för webbläsaren hur det ska se ut. Vad som ska vara en rubrik och vad som ska vara en länk eller en bild.

Dag 2: CSS

Cascading Style Sheets

CSS är ett sätt att berätta för webbläsare hur sidan ska se ut. Vilka färger det ska vara, vilka typsnitt som ska användas, hur långt det ska vara mellan olika grejer på sidan och så vidare.

JavaScript

Javascript är ett programmeringsspråk som vi kan använda för att lägga till funktioner på webbsidan.
