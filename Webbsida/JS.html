Javascript - funktioner på webbsidan

Vart lägger vi in javascript?

Javascipt lägger man in mellan script-taggar, <script> </script>. Script-taggarna kan man antingen placera uppe i head eller längst ner i body på webbsidan. Precis som vi skrev vår css i ett annat dokument kan vi skriva våran javascript i en separat fil så vi kan använda den på flera olika sidor. Detta gör vi genom att skriva <script> src="" </script> och skriva namnet på vår js-fil mellan "". T.ex. <script> src="minsida.js" </script>

Påverka din webbsida med javascript

För att påverka element på webbsidan behöver vi ett sätt att märka upp det vi vill påverka. Precis som vi använde class="" för att påverka html med css, så kan vi använda id="" för att påverka ett enskilt element. Skillnaden på class och id är att det bara får finnas ett id per webbsida, medan du kan använda samma class hur ofta du vill.

Nu kan vi påverka innehållet på webbsidan genom javascript commandot getElementById(). Säg att vi skapat ett id som heter id="mintext" då kan vi skicka javascript kommandon dit genom att använda document.getElementById("mintext"). Säg att vi vill ändra texten som står i <p id="mintext"> Jag heter Johan </p>. Då kan vi med hjälp av javascript skriva. <script> document.getElementById("mintext").innerHTML = "Jag är Batman."; </script> och texten kommer att ändras.

Knappar/Buttons

Ett vanligare sätt att använda javascript är med knappar. Om vi kopplar javascript till en knapp kan vi påverka när saker ska hända på webbsidan. Att klicka på en knapp brukar kallas för ett Event.

För att koppla javascript till en knapp använder man ofta funktioner. En funktion är ett litet program där man kan få flera saker att hända genom att bara skriva ett kort kommando.

Funktioner

För att skapa en funktion behöver vi hitta på ett namn på funktionen. Namnet behöver vara ett ord och det är bra om det beskriver vad funktinoen gör. Då blir det lättare när du kikar på programmet i framtiden.

T.ex. function bytaNamn() { }

Ok för att förklara: function skriver vi för att tala om att vi vill skapa en funktion. Sedan namnet vi vill kalla funktionen den vanliga parentesen ska sitta ihop med namnet och den använder man ifall vi vill skick in data som ska användas i funktionen. Sedan på slutet så har vi måsvingarna, den vågiga parentesen, innanför dessa skriver vi vårt program som ska köras när vi använder funktionen.

T.ex. function bytaNamn() { document.getElementById("mintext").innerHTML = "Jag är Batman."; }

Funktionen och knappen.

Nu behöver vi lägga till knappen på vår webbsida. För att göra detta lägger vi till html-taggen <button> Byt text </button> För att den ska fungera behöver vi också lägga till onclick="" och innanför "" skriver vi namnet på vår funktion, alltså:
<button onclick="bytaNamn()"> Byt text </button>

Vi kan också använda knappar för att byta ut bilder. Genom att sätta ett id på en img-tagg och sedan koppla en knapp till detta kan vi säga åt img-taggen vilken bild som ska visas.

T.ex. <img id="bild1" src="newworld.png" alt="">
<button onclick="bytBild()"> Byt bild </button>
unction bytBild() {
document.getElementById("bild1").src="LeagueOfLegends.png" }

Nu när vi trycker på knappen så kommer vår funktion säga åt img-taggen att den ska använda bildern som heter LeagueOfLegends.png istället för newworld.png som var där från början.
