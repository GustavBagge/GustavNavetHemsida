CSS att styla sin webbsida

Med CSS kan vi påverka hur vår webbsida ser ut. Vilka fäger, typsnitt och hur saker är placerade på sidan. Vi kommer även behöva några extra html-taggar t.ex. <div> för att påverka det vi vill.

För att påverka särskilda element kan vi även använda classer och id. Med class kan vi påverka flera olika element, t.ex. med class="katt" kan vi göra så att alla <p class="katt"> </p> får samma bakgrundsfärg eller samma färg på texten.

Med id kan vi påverka ett element per sida. T.ex. <p id="brutus"> </p>. Nu kan vi göra ändringar som bara påverkar just id="brutus".

Style i CSS

För att göra en ändring skapar vi ett dokument som ligger i samma mapp som vårt html-dokument och heter .css t.ex. hemsida.css

För att koppla ihop vårt nya dokument med vår html-sida behöver vi lägga in en kodrad under <head>. <link rel="stylesheet" href="hemsida.css"></p> rel talar om för webbsidan att det är styledokumentet och href talar om vart dokumentet ligger.

För att ändra något på ett element på webbsidan kan vi skriva p{ CSS-kod; } då ändras alla <p></p> på sidan som vi kopplat vår CSS till.

Genom att skriva .classnamn{ CSS-kod } kan vi påverka alla element med classen vi namngivit.
T.ex. .boxContent{ color: blue ;} som gör att alla element med klassen boxContent får blå text.

Genom att skriva #idnamn{ CSS-kod } kan vi påverka element med rätt id. Tänk på att det bara kan finnas ett unikt ID per webbsida.
T.ex. #rubrikJohan{font-family: wingdings; }

Exempel på style-element

Med width och height kan vi bestämma hur breda och höga saker ska vara. Detta kan vi ange i px som står för pixel och ger en bestämd bredd i antal pixlar. Ett annat sätt är att ange bredden i %. T.ex. width: 50%; då blir elementet vi stylar hälften så bred som sitt föräldraelement.

Vad är ett föräldraelement då? Det är förenklat sett html-taggen som omsluter elementets tagg. Ytterst finns html-taggen innanför den finns body-taggen osv. Då blir html-taggen förälder till body och om vi har satt att html ska ha bredden 100px och sedan säger vi att body ska ha bredd 50% så kommer den bli 50px bred. alltså 50% av 100px.

Med font-family kan vi ändra typsnittet på sidan, dvs. hur bokstäverna ser ut. Det finns några inbyggda som t.ex. Arial, Times new roman eller Verdana. Men man kan lägga till egna med hjälp av t.ex. Google fonts.

Med color: färgnamn eller kod; kan vi byta färg på texten på sidan och med background-color: färgnamn eller kod; kan vi byta färg på bakgrunden.

Med background-image: url( 'bildadress' ) kan vi lägga in en bild som bakgrund. DÅ behöver vi ofta styla och ändra så det täcker hela eller repeteras och bildar ett pussel. Här kan du läsa mer om hur du ska göra: https://www.w3schools.com/html/html_images_background.asp
